# knowledge-card-generator
## 知识点一图总结生成器（Wan万相妙思快闪赛参赛作品）

### 项目简介
本 Skill 可将任意学科知识点，通过「启动询问→信息提取→价值提炼→模块拆分→内容生成→生图执行」六步法，自动生成高信息密度知识卡片图片，适配复习、备考、笔记整理、知识分享等场景，支持跨学科复用。

### 核心功能
- 自动提取知识点核心定义、公式、易错点、记忆技巧
- 固定8模块结构化输出，逻辑清晰、重点突出
- 调用 `wan2.7-image-pro` 模型生成学术笔记风格知识卡片
- 支持任意学科，可批量生成知识卡片
